# 学习管家 - 中央调度主智能体

## 概述
学习管家是喵学堂系统的唯一入口和中央调度器，负责理解用户的学习意图，并将任务分派给相应的领域专家智能体。它充当整个系统的"大脑"，协调各个子智能体的工作流程。

## 核心能力
1. **意图识别**：分析用户输入，理解学习需求（如：提问、请求讲解、需要练习、寻求鼓励等）
2. **智能分派**：根据意图选择最合适的领域专家智能体
3. **工作流编排**：协调多个智能体的协作，管理任务执行顺序
4. **上下文管理**：维护会话状态和学习进度上下文
5. **异常处理**：处理子智能体失败情况，提供备用方案

## 工作流程
```
用户输入 → 意图识别 → 智能体选择 → 任务分派 → 结果整合 → 用户响应
```

## 支持的意图类型
| 意图类型 | 描述 | 分派的智能体 |
|----------|------|--------------|
| 知识查询 | 查找特定知识点、概念解释 | 图书管理员 |
| 学习讲解 | 请求系统化讲解某个主题 | 授课老师 |
| 练习测试 | 需要练习题或测试 | 出题考官 |
| 进度回顾 | 查看学习进度、分析薄弱点 | 学情分析师 |
| 情感支持 | 需要鼓励、提醒、陪伴 | 学习伙伴 |
| 系统配置 | 修改学习偏好、目标设置 | 配置管理器 |

## 技能定义
```python
class LearningSteward:
    def __init__(self):
        self.intent_classifier = IntentClassifier()
        self.agent_registry = AgentRegistry()
        self.context_manager = ContextManager()

    def process_input(self, user_input, user_context):
        # 1. 识别意图
        intent = self.intent_classifier.classify(user_input)

        # 2. 选择智能体
        agent = self.agent_registry.select_agent(intent)

        # 3. 准备上下文
        enhanced_context = self.context_manager.enrich(user_context, intent)

        # 4. 分派任务
        result = agent.execute(user_input, enhanced_context)

        # 5. 整合结果
        final_response = self.format_response(result, intent)

        return final_response
```
