# 向量的数量积

## 定义
- 向量的数量积（点乘/内积）定义为 a→·b→ = |a→||b→|cosθ，
  其中 θ 为两向量之间的夹角。
- 当 θ = 90° 时，a→⊥b→，则 a→·b→ = 0。
- 另一等价形式：a→·b→ = |a→||b→|cosθ = a₁b₁ + a₂b₂ + a₃b₃（若以直角单位向量表示）。

> 注：在直角单位向量基底 i→, j→, k→ 下，a→ = x₁i→ + y₁j→ + z₁k→，b→ = x₂i→ + y₂j→ + z₂k→。 

## 公式
- a→·b→ = |a→||b→|cosθ
- a→·b→ = x₁x₂ + y₁y₂ + z₁z₂
- i→·j→ = 0; i→·i→ = 1; j→·j→ = 1; k→·k→ = 1
- i→·k→ = 0; j→·k→ = 0
- 坐标展开：a→ = x₁i→ + y₁j→ + z₁k→, b→ = x₂i→ + y₂j→ + z₂k→
- a→·b→ = (x₁x₂ + y₁y₂ + z₁z₂)
- W = |F→|·|S→| = |F→|cosθ·|S→|（力×位移）

## 性质
- 线性性：a→·(b→+c→) = a→·b→ + a→·c→
- 分配性：(a→+b→)·c→ = a→·c→ + b→·c→
- 对称性：a→·b→ = b→·a→
- 与模长和夹角的关系：a→·b→ = |a→||b→|cosθ

## 例题
- 例1：设 a = (1, 2, 3)，b = (4, -5, 6)，求 a·b。
  解：a·b = 1×4 + 2×(-5) + 3×6 = 4 - 10 + 18 = 12。
- 例2：若 |a| = 3, |b| = 4, 夹角 θ = 60°，求 a·b。
  解：a·b = |a||b|cosθ = 3×4×cos60° = 12×0.5 = 6。

## 注意事项
- 当 θ = 90° 时 a→·b→ = 0。
- 向量的分解要注意单位向量的正交性，i→·j→ = 0 等。
- 物理应用中，功 W 的公式可写成 W = F·S = |F||S|cosθ。
- 坐标展开时，请区分分量乘积与向量模长的乘积。
