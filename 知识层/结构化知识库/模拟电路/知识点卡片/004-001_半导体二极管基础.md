# 半导体二极管基础

## 1. 半导体材料

### 本征半导体
- 纯净的半导体材料
- 常见材料：**硅(Si)**、锗(Ge)

### 掺杂半导体
- 在本征半导体中掺入杂质原子
- 形成P型半导体（空穴多）和N型半导体（电子多）

## 2. 二极管结构与符号

### 结构
- 由P型半导体和N型半导体结合形成**PN结**
- 引出两个电极：
  - **阳极(Anode, A)**：连接P区
  - **阴极(Cathode, K)**：连接N区

### 电路符号
```
    阳极(A)        阴极(K)
      │              │
      ├───|>|───
      │              │
```
箭头方向表示正向导通时的电流方向

## 3. 单向导电性

### 正向偏置（正偏）
- 阳极接高电位，阴极接低电位
- **PN结导通**，电流可以流过

### 反向偏置（反偏）
- 阳极接低电位，阴极接高电位
- **PN结截止**，几乎无电流（仅有极小反向漏电流）

## 4. 核心要点

| 状态 | 偏置方式 | 导通情况 |
|------|----------|----------|
| 正偏 | 阳极 > 阴极 | 导通 ✓ |
| 反偏 | 阳极 < 阴极 | 截止 ✗ |

## 5. 常见应用

- **整流电路**：将交流电转换为直流电
- **检波电路**：从调制信号中提取信息
- **限幅电路**：限制信号幅度
- **开关电路**：利用其开关特性

---

> **记忆口诀**：正偏导通，反偏截止；阳极进，阴极出。
