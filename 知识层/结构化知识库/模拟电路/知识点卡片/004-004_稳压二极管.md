# 稳压二极管（齐纳二极管）

## 1. 工作原理

稳压二极管是一种特殊的二极管，工作在**反向击穿区**，利用反向击穿时电压基本恒定的特性实现稳压功能。

```
        反向特性
I ↑
  │
  ├──────────────→ U
  │
  │      击穿区
  │     ╱│
  │    ╱ │← Uz 基本恒定
  │   ╱  │
```

## 2. 主要参数

### ① 稳定电压 U_Z
- 稳压管正常工作时两端的**稳定电压值**
- 由制造工艺决定，具有固定值
- 常见值：3.3V、5.1V、6.2V、9.1V、12V、15V等

### ② 稳定电流 I_Z
- 稳压管正常工作时的**参考电流值**
- 电流过小：未进入稳压区，稳压效果差
- 电流过大：接近最大电流，功耗增加

### ③ 最大稳定电流 I_ZM
- 稳压管允许通过的**最大反向电流**
- 超过此值可能因过热而损坏

### ④ 动态电阻 r_Z
- 稳压管在稳压区的**动态电阻**
- 定义：r_Z = ΔU_Z / ΔI_Z
- **r_Z 越小，稳压性能越好**
- 理想稳压管的 r_Z = 0

### ⑤ 最大允许耗散功率 P_ZM
- 稳压管正常工作时允许的**最大功耗**
- 计算公式：**P_ZM = U_Z × I_ZM**
- 超过此值会导致器件过热损坏

## 3. 参数关系

```
P_ZM = U_Z × I_ZM

动态电阻 r_Z = ΔU_Z / ΔI_Z
（越小越好，通常在几欧姆到几十欧姆）
```

## 4. 典型应用电路

### 简单稳压电路
```
        R（限流电阻）
Ui ───┬────//\/\────┬─── Uo（稳定输出）
      │               │
     ═══             ═╪═ 稳压管Dz
      │               │
     GND             GND
```

### 工作原理
1. 输入电压 Ui 经限流电阻 R 降压
2. 稳压管 Dz 工作在反向击穿区
3. 输出电压 Uo ≈ U_Z（稳定）

### 限流电阻计算
```
R = (Ui - Uz) / Iz
```
其中 Iz 应满足：I_Z < Iz < I_ZM

## 5. 使用注意事项

1. **必须串联限流电阻**，否则电流过大会烧毁稳压管
2. 稳压管工作在**反向偏置**状态
3. 输入电压 Ui 必须 **> U_Z**，否则无法稳压
4. 负载电流变化时，稳压管电流会反向补偿

## 6. 稳压管 vs 普通二极管

| 特性 | 普通二极管 | 稳压二极管 |
|------|------------|------------|
| 工作区 | 正向导通区 | 反向击穿区 |
| 用途 | 整流、开关 | 稳压、基准电压 |
| 反向电压 | 不允许超过U_RM | 专门设计用于击穿区 |
| 功耗 | 较小 | 较大（需散热考虑）|
