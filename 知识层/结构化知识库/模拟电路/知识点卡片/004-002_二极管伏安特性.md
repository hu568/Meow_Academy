# 二极管的伏安特性

## 1. 伏安特性曲线

二极管的伏安特性描述了通过二极管的电流 I 与两端电压 U 之间的关系。

```
I ↑
  │
  │        正向特性          反向特性
  │    ╱
  │   ╱ 导通区
  │  ╱
  │ ╱ 死区
  ├────────┬────────────→ U
  │       0.6V(硅)
  │
  │        反向漏电流
  └─────────────────────→
           击穿电压
```

## 2. 正向特性

### 死区（截止区）
- 正向电压较小时，电流几乎为零
- 这个电压范围称为**死区**

### 死区电压
| 材料 | 死区电压 |
|------|----------|
| 硅(Si) | 约 **0.6V** |
| 锗(Ge) | 约 **0.2V** |

### 导通电压
当正向电压超过死区电压后，二极管导通：

| 材料 | 导通电压范围 |
|------|-------------|
| 硅(Si) | **0.6V ~ 0.7V** |
| 锗(Ge) | **0.2V ~ 0.3V** |

### 导通后特性
- 导通后，电流随电压**急剧增加**
- 管压降基本保持在导通电压附近
- 硅管：约0.7V，锗管：约0.3V

## 3. 反向特性

### 反向截止区
- 反向电压在一定范围内，仅有极小的**反向漏电流**
- 硅管的反向电流远小于锗管

### 反向击穿区
- 当反向电压超过一定值（击穿电压）时
- 反向电流急剧增大，二极管可能被损坏

## 4. 硅管 vs 锗管对比

| 特性 | 硅管 | 锗管 |
|------|------|------|
| 死区电压 | 0.6V | 0.2V |
| 导通电压 | 0.6~0.7V | 0.2~0.3V |
| 反向漏电流 | 很小（nA级） | 较大（μA级） |
| 温度稳定性 | 好 | 较差 |
| 最高工作温度 | 150~200°C | 70~90°C |
| 活跃度 | 较低 | **较高** |

> **笔记提示**：锗比硅活跃（更容易导通，死区电压更低）

## 5. 温度影响

- 温度升高 → 死区电压降低
- 温度升高 → 反向漏电流增大
- 硅管的温度稳定性优于锗管

## 6. 工程近似模型

### 理想模型（开关模型）
- 正偏：短路（U_D = 0）
- 反偏：开路（I_D = 0）

### 恒压降模型
- 正偏：U_D = 0.7V（硅）或 0.3V（锗）
- 反偏：开路

### 折线模型
- 考虑导通电阻，更精确

---

> **记忆口诀**：硅管死区零点六，锗管死区零点二；锗比硅活跃，硅比锗稳定。
