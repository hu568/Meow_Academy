# 反向工作电压

## 定义

二极管在反向偏置状态下工作时，两端承受的电压。

## 最高反向工作电压 U_RM

### 定义
保证二极管安全工作时，允许施加的**最大反向电压**。

### 符号
- **U_RM**：Reverse Maximum Voltage
- 有时也记为 **V_RWM** 或 **PIV**（Peak Inverse Voltage）

### 与击穿电压的关系
```
U_RM = (1/2 ~ 2/3) × U_BR
```
其中 U_BR 为反向击穿电压

### 超过后果
- 进入反向击穿区
- 反向电流急剧增大
- 功耗剧增，可能导致**热击穿**（永久损坏）

## 选型原则

实际反向电压应满足：
```
U_反向峰值 ≤ (0.5 ~ 0.7) × U_RM
```

## 整流电路中的反向电压

| 整流电路类型 | 最大反向电压 |
|-------------|-------------|
| 半波整流 | U_RM ≥ √2 × U_2 |
| 全波整流 | U_RM ≥ 2√2 × U_2 |
| 桥式整流 | U_RM ≥ √2 × U_2 |

（U_2为变压器副边电压有效值）
