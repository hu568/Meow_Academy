# 稳压二极管

## 定义

一种特殊的硅二极管，工作在**反向击穿区**，利用反向击穿时电压基本恒定的特性来实现**稳压功能**。

又称**齐纳二极管**（Zener Diode）。

## 工作原理

### 反向击穿机制
1. **齐纳击穿**（低电压，< 6V）：强电场直接破坏共价键
2. **雪崩击穿**（高电压，> 6V）：载流子碰撞电离连锁反应

### 稳压特性
- 击穿后，电流在很大范围内变化时
- 两端电压**基本保持不变**
- 这个恒定电压即为**稳定电压 U_Z**

## 电路符号

```
    阴极(K)      阳极(A)
      │            │
      ├───|>|──//\/\───
      │    带拐弯的符号
```

或简化为：
```
      ───|>|───
         ╱
```

## 主要参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 稳定电压 | U_Z | 稳压值 |
| 稳定电流 | I_Z | 正常工作参考电流 |
| 最大稳定电流 | I_ZM | 允许的最大电流 |
| 动态电阻 | r_Z | r_Z = ΔU_Z/ΔI_Z |
| 最大耗散功率 | P_ZM | P_ZM = U_Z × I_ZM |

## 典型应用

### 简单并联稳压电路
```
Ui ──[R]──┬── Uo
           │
          [Dz]
           │
          GND
```

### 应用场景
- 电源稳压
- 过压保护
- 基准电压源
- 电平钳位

## 使用注意

1. **必须串联限流电阻**
2. 工作在**反向偏置**
3. 输入电压必须 **> U_Z**
4. 注意功耗和散热

## 与普通二极管的区别

| 特性 | 普通二极管 | 稳压二极管 |
|------|------------|------------|
| 工作区 | 正向导通 | 反向击穿 |
| 设计目的 | 整流/开关 | 稳压 |
| 反向耐压 | 有限制 | 专门设计用于击穿 |
| 功耗 | 较小 | 较大 |
