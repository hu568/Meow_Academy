# 整流电流

## 定义

二极管作为整流元件时，允许通过的**正向电流**。

## 最大整流电流 I_F（I_OM）

### 定义
二极管长期运行时允许通过的**最大正向平均电流**。

### 符号
- **I_F**：Forward Current
- **I_OM**：Operating Maximum Current

### 限制因素
- **发热**：电流过大会导致PN结温度过高
- **引线熔断**：过大电流可能熔断引线
- **器件烧毁**：严重过热导致器件永久损坏

### 选型原则
实际工作电流应满足：
```
I_实际 ≤ (0.5 ~ 0.7) × I_F
```

## 其他电流参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 正向电流 | I_F | 正向导通时的电流 |
| 浪涌电流 | I_FSM | 允许的非重复峰值电流 |
| 反向电流 | I_R | 反向偏置时的漏电流 |

## 应用注意

- 整流电路中需根据负载电流选择二极管
- 大电流场合可采用多个二极管并联（需均流措施）
- 高频整流需考虑开关损耗
