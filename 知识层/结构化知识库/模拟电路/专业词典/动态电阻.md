# 动态电阻

## 定义

器件在工作点附近，电压微小变化量与电流微小变化量之比：

```
r = ΔU / ΔI
```

也称为**交流电阻**或**小信号电阻**。

## 稳压二极管的动态电阻 r_Z

### 定义
稳压管在稳压区工作时：
```
r_Z = ΔU_Z / ΔI_Z
```

### 物理意义
- 反映稳压管维持电压恒定的能力
- **r_Z 越小，稳压性能越好**
- 理想稳压管的 r_Z = 0

### 典型值
- 几欧姆到几十欧姆
- 与稳压值和工作电流有关

## 与静态电阻的区别

| 类型 | 定义 | 用途 |
|------|------|------|
| 静态电阻 | R = U/I | 直流分析 |
| 动态电阻 | r = ΔU/ΔI | 交流/小信号分析 |

## 计算示例

若稳压管在 I_Z = 10mA 时 U_Z = 6V，
当电流变化 ΔI_Z = 5mA 时，电压变化 ΔU_Z = 0.05V

则动态电阻：
```
r_Z = 0.05V / 0.005A = 10Ω
```

## 工程意义

- 动态电阻影响稳压电路的**稳压系数**
- 负载变化时，动态电阻越小，输出电压越稳定
- 选择稳压管时应优先选择 r_Z 小的型号

## 其他应用

动态电阻概念也适用于：
- 二极管的正向导通电阻
- 晶体管的输入/输出电阻
- 其他非线性器件的小信号分析
